I1 verify_delegation_permission() 권한 검증 — 위임 SB H3 알고리즘 5단계 검증

SELECT * FROM verify_delegation_permission( p_actor_id => 123, p_action => 'submit_review', p_series_id => 456 ); -- 반환: (ok, mode, delegation_id, reason) -- ok=TRUE, mode='delegated', delegation_id='D-2432', reason=NULL

알고리즘 5단계

Step검증결과
1작품의 author 조회없으면 reason='series_not_found'
